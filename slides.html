Browserify

https://npmjs.org/package/browserify

Browserify

Exactly the same as writing Node on the server.

You get require and module.exports. Things like process, utils and http all still work the way you expect in Node code.

And best of all, most NPM packages just work:

var marked = require('marked')

------------

Almost every Node.js API now has an equivalent browser API.

Modules

  • browserify bundles Node code up for the browser
  • context-server streams changes between server and browser
  • brfs transforms fs.readFileSync to inline
  • become provides partial page updates
  • dom-behavior allows us to hook DOM events
  • former populates form elements and gets changes
  • dom-morph animates between two DOM elements

github.com/mmckegg/realtime-blog-example-with-browserify

More Modules

  • supervisor automatically reloads Node app when code changes
  • level runs LevelDB inside your Node process
  • contextdb indexing and view creation from LevelDB
  • routes-router match incoming requests and route to functions
  • rincewind pure html based templating engine
